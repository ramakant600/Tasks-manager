# **App Name**: TaskMaster

## Core Features:

- Task Management: Add, delete, and toggle the completion status of tasks. Tasks are stored in local storage.
- Task Filtering: Filter tasks to display all, completed, or pending tasks.
- Theme Switching: Toggle between a light and dark theme for the application.

## Style Guidelines:

- Primary color: Neutral grays for a clean interface.
- Secondary color: A calming blue (#3498db) for interactive elements.
- Accent: Green (#2ecc71) to indicate task completion.
- Clear and readable font for task descriptions.
- Simple and intuitive icons for task actions (add, delete, complete).
- Clean and organized layout for easy task management.
- Subtle transitions for adding and removing tasks.
